🚀 Dynamic Form Builder (Full Stack)

A full stack web application where users can register, login, create custom forms, manage them, and share forms publicly.

🌐 Live Demo: https://dynamic-form5.netlify.app/

📌 Project Overview

Dynamic Form Builder is a MERN stack application that allows authenticated users to:

🔐 Register & Login securely

📝 Create fully customizable forms

📋 View all created forms

✏️ Edit or delete forms

🔗 Share a specific form via public link

This project demonstrates strong understanding of:

Authentication & Authorization

REST API architecture

CRUD operations

State management in React

MongoDB data modeling

Full-stack integration
